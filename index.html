About New Kernel


The New Kernel's goal is to make OS development more modern by encouraging people to use Rust over older low-level languages. Our main project is Novusk, it is a kernel written in Rust that is intended for many uses. We have other projects that can be used for OS/bare-metal projects, like Novus Boot a bootloader for Novusk and any device that supports UEFI.


You can see our projects and even help on Github. New Kernel on Github. New contributors are always welcome!
