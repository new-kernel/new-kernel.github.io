About Novusk:


Novusk is a kernel inspired by Linux written in Rust intended for OS and bare-metal app development, but can be used for many other things too. Novusk is opensource so it can also be used as a learning resource for people wanting to modernize OS development and use a new language, you can find New Kernel's fork of Novusk here and the main repository can be found here. As of right now Novusk is developing its fourth version and improving its third, has released several beta versions.

Novusk v3 2022/4/23

Even though Novusk v3 just recently came out it's starting to look like it shouldn't be used anymore or needs almost a complete rewrite because of a few problems. Lots of Novusk's dependencies are unmaintained and don't work with the current version of Rust (1.60 - 1.62), another thing is linker errors caused by drivers getting mixed together. Fixing unmaintained dependencies can be done by implementing them manually and fixing linker errors could be done with a bit of reorganizing. Doing a rewrite is probably the best thing to do to fix this problem and prevent it from happening again, This (#22) is an issue to keep track of the problem

Novusk v3 2022/1/12

Novusk v3 has been released! It is the biggest major version released so far, it improves many features and adds many more. This new version added architecture support for ARM (32 bit), RISCV32, and Xtensa (Xtensa is experimental), added drivers: graphics, basic networking, basic disk/sd reading, and firmware accessing (mailbox). System calls and user libraries like LibC have been greatly improved which makes Novusk much more accessible and easier to use. This tag has a list of all things that have been added.

Novusk v3 is being moved to this branch for maintaining and future updates, most minor or really minor updates should add features from this issue which is a TODO list for Novusk v3, and the master branch will be used for developing future versions and use parts of the "v3" branch if needed.
