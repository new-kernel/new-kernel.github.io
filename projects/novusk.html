About Novusk:


Novusk is a kernel inspired by Linux written in Rust intended for OS and bare-metal app development, but can be used for many other things too. Novusk is opensource so it can also be used as a learning resource for people wanting to modernize OS development and use a new language, you can find New Kernel's fork of Novusk here and the main repository can be found here.
